# Editors

*   [vscode][]: VSCode
*   [vscode-mdx-preview][]: MDX Preview for VS Code
*   [vim][]: Vim
*   [sublime][]: Sublime
*   [JetBrains IntelliJ/WebStorm][jetbrains]
