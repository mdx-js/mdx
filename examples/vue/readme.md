# Vue + MDX

> [Try it on CodeSandbox](https://codesandbox.io/s/github/mdx-js/mdx/tree/main/examples/vue)

Install:

```sh
npm install
```

Then, to start a development server:

```sh
npm start
```

Alternatively, to build for production and serve the resulting `dist/`:

```sh
npm run build
npx serve dist/
```
